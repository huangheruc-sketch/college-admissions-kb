# FIRST Robotics Sources

## 官方来源

| 来源 | URL | 用途 |
|------|-----|------|
| FIRST 官网 | https://www.firstinspires.org/ | 赛事概况 |
| FRC 项目页 | https://www.firstinspires.org/robotics/frc | FRC 概述 |
| FTC 项目页 | https://www.firstinspires.org/robotics/ftc | FTC 概述 |
| FIRST Team Search | https://www.firstinspires.org/team-event-search | 查找团队 |
| FRC Awards | https://www.firstinspires.org/robotics/frc/awards | 奖项说明 |
| FIRST Scholarships | https://www.firstinspires.org/alumni/scholarships | 奖学金 |
| FRC Game Manual | https://www.firstinspires.org/resource-library/frc/competition-manual-qa-system | 赛季规则 |

## 社区数据来源

| 来源 | URL | 用途 |

| The Blue Alliance | https://www.thebluealliance.com/ | FRC 历年赛事数据、成绩、视频 |

## 数据说明

- FRC 团队数量和奖学金数据引自 FIRST 官方公开信息
- 具体团队获奖历史可在 The Blue Alliance 中查询验证
- 中国赛区团队信息可能随年度更新，建议以 FIRST 官网最新数据为准
