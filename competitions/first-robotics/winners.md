# FIRST Robotics 标志性奖项与高水平案例

> FIRST Robotics 的最高荣誉是 Impact Award（原 Chairman's Award），而非淘汰赛冠军。以下按奖项类型整理高水平案例。
>
> **注意**：FRC 的奖项数据分散在区域赛和锦标赛中，以下仅收录 Championship 级别的标志性案例。部分信息来自 FIRST 官方及公开报道，未逐一验证每个细节。

---

## Impact Award（原 Chairman's Award）

Impact Award 是 FIRST 的最高荣誉，表彰团队对推广 STEM 教育和工程文化的长期社区影响——不评估机器人竞技表现。

### 知名多次获奖团队

| 团队 | 说明 |
|------|------|
| **Team 1114 — Simbotics**（加拿大 St. Catharines） | FRC 传奇团队之一，多次获得 Chairman's Award 和竞技冠军 |
| **Team 341 — Miss Daisy**（Wissahickon, PA） | 多次 Chairman's Award 获奖团队 |
| **Team 1678 — Citrus Circuits**（Davis, CA） | 竞技与文化双强，广泛的社区推广项目 |
| **Team 254 — The Cheesy Poofs**（San Jose, CA） | FRC 历史上最成功的竞技团队之一，多次 Championship 冠军 |

> 获得 Impact Award 的团队通常有多年的 STEM 推广历史——建立社区 FLL 团队、举办工程夏令营、开发开源教育资源等。

---

## Dean's List Award

Dean's List 是 FRC 中面向个人学生的最高荣誉（每支 Finalist 团队可提名最多 2 名学生，全球约 20 名最终获奖）。获奖标准：对团队和社区的杰出领导力与贡献。

> Dean's List 在大学申请中有较高辨识度，尤其对于 MIT、Georgia Tech、WPI 等工程强校。

## 竞技方向的高水平表现

FRC 竞技奖不是单一"冠军"——比赛以 3 队联盟制进行，联盟冠军 = 3 支团队共同获奖。

### 近年 Championship 竞技强队（公开信息）

| 团队 | 说明 |
|------|------|
| **Team 254 — The Cheesy Poofs** | 多年 Championship 冠军/决赛，以软件和策略著称 |
| **Team 1114 — Simbotics** | 多年 Championship 冠军/决赛，以机械设计著称 |
| **Team 2056 — OP Robotics** | 加拿大强队，多次 Championship 级别表现 |
| **Team 1678 — Citrus Circuits** | 近年 Championship 常客 |

## 项目方向参考

FRC 每年挑战不同，但高水平团队的核心工程能力相对稳定：

| 技术方向 | 说明 |
|----------|------|
| 机械设计 | 底盘设计（swerve drive）、机构设计（intake、shooter、climber） |
| 软件/自动化 | 路径规划、计算机视觉（AprilTag 识别）、自动导航 |
| 控制系统 | PID 调参、运动学建模、传感器融合 |
| 策略分析 | 赛场数据 scouting、联盟选择策略 |
| 项目管理 | 甘特图、迭代设计流程、团队分工 |

> 这些技术能力与大学工程课程高度对标——FRC 经历在工程方向申请中的叙事力很强。
