# FIRST Robotics — FIRST Robotics Competition (FRC)

> 全球规模最大的高中生机器人工程竞赛，由 FIRST（For Inspiration and Recognition of Science and Technology）主办。本页以 FRC 为主，FTC 作为补充说明。

## Quick Facts

| 项目 | 内容 |
|------|------|
| 全称 | FIRST Robotics Competition (FRC) |
| 主办方 | [FIRST](https://www.firstinspires.org/)（创始人 Dean Kamen） |
| 创立年份 | 1992（FRC）；2005（FTC） |
| 参赛年级 | 9–12 年级（14–18 岁） |
| 参赛形式 | **团队制**（每队 25–50+ 名学生 + 成人导师） |
| 每年参赛队伍 | FRC ~3,800+ 支（全球）；FTC ~8,000+ 支 |
| 参赛国家/地区 | 30+ |
| 赛季周期 | 1 月发布挑战 → 6 周造机 → 3–4 月区域赛 → 4 月锦标赛 |
| 最高荣誉 | Chairman's Award（现更名 Impact Award，表彰团队对社区的持续影响） |
| 奖学金 | 每年发放超过 $80,000,000 |
| 官网 | [firstinspires.org](https://www.firstinspires.org/) |

## 适合什么类型的学生？

FIRST Robotics 与 ISEF/STS 的核心区别：**团队工程项目，强调动手造机 + 协作 + 领导力**，而非个人科研论文。

- **动手能力强、喜欢工程/制造的学生**：需要设计、制造、编程一台 120+ 磅的工业级机器人
- **团队协作型**：FRC 强调"Coopertition"（竞合），评审关注团队文化与社区影响
- **有项目管理/领导力兴趣的学生**：团队需要自行筹款、品牌运营、社区推广
- **编程/控制系统方向的学生**：机器人自动导航、视觉识别、运动控制是核心技术模块
- **商业/策略思维的学生**：商业计划、赞助沟通、项目管理也是评审维度

### FTC 作为入门路径

**FIRST Tech Challenge (FTC)** 是 FRC 的"缩小版"：机器人更小（~42 lbs）、团队更小（~15 人）、成本更低（~$3,000–$5,000 vs FRC 的 $20,000+）。对于缺乏导师资源或资金的学校，FTC 是很好的起步选择。FTC 的经验可直接迁移到 FRC。

## 项目/产出形态

- **一台完整的机器人**：每年 1 月 FIRST 发布新赛季挑战（全新游戏规则），团队有 **6 周时间**从零设计、制造并测试机器人
- **技术文档**：控制系统代码（通常开源在 GitHub）、CAD 设计图、工程笔记本
- **商业计划 / 品牌 / 社区影响**：Impact Award 要求提交 essay + 视频 + 现场答辩
- **赛场对抗**：3v3 联盟对抗，策略选择与联盟合作是关键

## 与大学申请的关联

FIRST Robotics 在美国大学申请中有独特价值，尤其体现在工程方向：

- **MIT**：多次公开提及 FIRST，招生办有 FIRST 校友；MIT 的 2.007 课程（机械工程入门）与 FRC 精神高度契合
- **WPI**：提供 FIRST 专项奖学金
- **全美 200+ 所大学** 提供 FIRST 相关奖学金，总额每年超过 $80M
- **展示维度丰富**：不仅是技术能力，还能展示领导力、团队协作、项目管理、社区服务——这些正是顶尖大学最看重的"spike"
- **Dean's List Award**（个人奖）和 **Impact Award**（团队奖）在申请中具有很高辨识度

> FIRST 的独特价值：它不只是一个竞赛成绩——**参与过程本身**（组建团队、筹集资金、设计工程方案、社区推广）就构成了极其丰富的申请素材。
