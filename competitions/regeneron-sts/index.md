# Regeneron STS — Science Talent Search

> 美国历史最悠久、最负盛名的高中生科学与数学研究竞赛，由 Society for Science 主办，Regeneron 冠名赞助。

## Quick Facts

| 项目 | 内容 |
|------|------|
| 全称 | Regeneron Science Talent Search |
| 主办方 | [Society for Science](https://www.societyforscience.org/) |
| 冠名赞助 | [Regeneron Pharmaceuticals](https://www.regeneron.com/)（2017 起；此前为 Intel、Westinghouse） |
| 创立年份 | 1942（前身 Westinghouse Science Talent Search） |
| 参赛资格 | 美国高中 12 年级学生（Senior year）；少量国际学校学生可参与 |
| 每年申请人数 | ~2,500–2,700 |
| Top 300 Scholars | 从全部申请者中选出 |
| Top 40 Finalists | 从 300 Scholars 中选出 |
| 总奖金 | ~$1,800,000+ |
| 最高奖项 | 第一名 $250,000 |
| 官网 | [societyforscience.org/regeneron-sts](https://www.societyforscience.org/regeneron-sts/) |

## 适合什么类型的学生？

STS 与 ISEF 的核心区别：**STS 强调独立研究深度 + 学术潜力的全面评估**，而不仅是项目本身。

- **独立科研能力强的学生**：需要提交一篇完整的原创研究报告（类似于本科毕业论文水平）
- **学术全面型**：除研究外，还评估领导力、社区参与、课外活动等（全面审核，holistic review）
- **数学/理论方向的学生**：STS 对纯数学、理论物理等非实验性研究的包容度高于 ISEF
- **有导师资源的学生**：多数 Finalist 有大学实验室研究经历

## 项目/产出形态

- 提交一份 **20 页以内的原创研究报告**（科学论文格式）
- 涵盖所有 STEM 领域：数学、物理、化学、生物、计算机科学、工程、社会科学等
- 不限于实验研究：理论证明、计算建模、统计分析、工程设计均可
- Finalist 阶段需在华盛顿 D.C. 向评委进行面对面展示

## 与大学申请的关联

STS 是美国大学申请中**最具分量的 STEM 竞赛荣誉之一**，尤其是 Top 300 Scholar 和 Top 40 Finalist。

- **Scholar（Top 300）**：已是强有力的申请标签，标志着研究能力获得国家级认可
- **Finalist（Top 40）**：极高含金量，顶尖大学招生官普遍熟悉
- **知名校友**：STS 历史上产生了 13 位诺贝尔奖得主、23 位麦克阿瑟天才奖得主、14 位美国国家科学奖章得主
- **MIT / Caltech / Stanford / Harvard**：STS Finalist 在这些学校的录取中有显著优势

> STS 的独特价值在于它不仅评估研究成果，还评估学生作为"未来科学家"的整体潜力——这与顶尖大学的评估逻辑高度一致。
