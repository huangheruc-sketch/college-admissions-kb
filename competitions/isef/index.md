# ISEF — Regeneron International Science and Engineering Fair

> 全球规模最大的高中生科学与工程竞赛，由 Society for Science 主办，Regeneron 冠名赞助。

## Quick Facts

| 项目 | 内容 |
|------|------|
| 全称 | Regeneron International Science and Engineering Fair |
| 主办方 | [Society for Science](https://www.societyforscience.org/) |
| 冠名赞助 | [Regeneron Pharmaceuticals](https://www.regeneron.com/) |
| 参赛年级 | 9–12 年级（14–19 岁） |
| 每年入围人数 | ~1,700 名 |
| 参赛国家/地区 | 60+ |
| 总奖金 | ~$9,000,000+ |
| 最高奖项 | George D. Yancopoulos Innovator Award ($100,000，2025 起；此前为 $75,000) |
| 竞赛类别 | 22 个学科类别 |
| 官网 | [societyforscience.org/isef](https://www.societyforscience.org/isef/) |

## 与大学申请的关联

ISEF 是顶尖大学申请中最具分量的课外活动之一。入围 ISEF（Finalist）本身即是高含金量的成就；获得 Grand Award 或 Top Award 的学生在 STEM 方向申请中具有显著优势。

- **MIT**：招生页面明确提及 ISEF 等竞赛作为课外活动评估的参考
- **Caltech**：高度重视 STEM 研究经历，ISEF 获奖者在申请池中有突出表现
- **Stanford / Harvard**：虽不单独强调竞赛，但 ISEF Top Award 获奖者被录取的公开案例很多
